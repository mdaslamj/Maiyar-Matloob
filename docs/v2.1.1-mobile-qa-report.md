**Sprint:** Mobile Responsive Refinement

## Breakpoints Verified (CSS)

| Width | Layout |
|-------|--------|
| ≥1024px (desktop) | `repeat(auto-fit, minmax(320px, 1fr))` dashboard grid |
| 768–1023px (tablet) | 2-column dashboard, 20px padding |
| ≤767px (mobile) | Single column, ordered cards, 16px padding |
| ≤480px | Compact questionnaire navigation and ranked lists |

## Regression Risks

| Risk | Mitigation |
|------|------------|
| Long Urdu labels wrap | `overflow-wrap: anywhere` on cards/buttons |
| Dashboard order on desktop | `order` only affects mobile breakpoint |
| Print layout | Print CSS block untouched except inherited spacing |
