# Version 2.0.0 Release Notes

**Release date:** 2026-07-02  
**Product:** Maiyaar (معیار | محاسبۂ نفس)

Version 2.0 completes the personal self-development platform and delivers a production polish pass. No assessment content, scoring methodology, or protected engine calculations were changed.

---

## Highlights

### Growth Intelligence (v2 foundation)

- Trend, Action, and Feedback engines
- سفرِ ترقی (Growth Journey) dashboard
- Reassessment planner (7 / 14 / 30 / 60 days — planning only)

### Continuous Muhasabah

- **مرکزی صفحہ** — personal home dashboard
- Reflection journal (optional, score-independent)
- **ترقی کا وقت خط** — unified timeline of assessments and reflections
- Personal insights and reassessment planning views

### Version 2.0 Polish

- Unified Urdu terminology across screens
- Accessibility: skip link, landmarks, ARIA, keyboard-friendly toggles, reduced motion
- Responsive and print improvements
- Growth pipeline caching (`getGrowthBundle()`)
- Storage safety with Urdu user messages for corruption/quota failures
- XSS-safe rendering for journal content in timeline

---

## Unchanged (Protected)

- `questionnaire.json`
- Assessment and scoring methodology
- Report calculations
- History snapshot schema
- Trend, Action, and Feedback calculations

---

## Not Included

- Firebase sync (infrastructure present, disabled)
- Authentication
- Analytics
- Community / social features

---

## Upgrade Notes

- Existing v1.0.x localStorage data remains compatible
- Application version displayed as **ورژن 2.0.0**
- Welcome history button renamed to **ترقی کا وقت خط** for consistency
