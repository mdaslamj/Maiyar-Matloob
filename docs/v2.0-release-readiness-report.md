## Known Limitations

1. **No automated test suite** — manual regression only (see `docs/qa-checklist.md`)
2. **Native dialogs** — delete confirmations use `window.confirm` / `alert`
3. **No URL routing** — browser back button does not map to in-app screens
4. **Single-device** — no sync; clearing browser data removes all history
5. **Journal linked to latest assessment** — home form auto-links optional reflection to most recent snapshot
6. **Firebase disabled** — cloud adapters throw if enabled without full setup
7. **Offline fonts** — Noto Naskh Arabic loaded from Google Fonts CDN
