# Version 1.0.1 Release Notes

**Product:** Maiyaar (معیار | محاسبۂ نفس)  
**Release type:** Production stabilization (PATCH)  
**Date:** 2026-07-02  
**Tag:** `v1.0.1`

---

## Summary

Version 1.0.1 stabilizes the production experience without changing assessment content or scoring methodology. Users receive accurate section reports, reliable session/report persistence, clear Urdu feedback, and complete printable weak-question analysis.

---

## What Changed for Users

1. **Accurate section scores** in the detailed report (previously some sections showed 0% incorrectly).
2. **Consistent strongest and growth areas** — cards match their explanation text.
3. **Clear error message** if the questionnaire fails to load.
4. **Validation message** when trying to continue without selecting an answer.
5. **Resume after refresh** — continue or restart an in-progress assessment.
6. **Report restored after refresh** — no need to retake the questionnaire after completion.
7. **Urdu navigation and level labels** throughout the questionnaire and report.
8. **Complete print output** for weak-question analysis sections.
