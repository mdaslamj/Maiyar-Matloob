# User Flow

## First Visit

1. Welcome screen loads questionnaire
2. User reads scale legend → **شروع کریں**
3. 50-question flow with save/resume
4. Completion → printable report
5. Snapshot saved to history

## Returning User

1. Welcome → **مرکزی صفحہ** (if assessments exist)
2. Home shows last assessment, priorities, insights, reassessment plan
3. **جائزہ جاری رکھیں** / **نیا جائزہ شروع کریں**
4. Optional reflection journal entry

## Continuous Muhasabah Loop

```
Complete assessment
    → Report
    → Home dashboard (summary + priorities)
    → Reflection journal (optional)
    → Growth timeline (assessments + reflections)
    → سفرِ ترقی (trends + action plan + feedback)
    → Reassessment plan (informational date only)
    → Next assessment when ready
```

## Navigation Map

| Screen | Entry points |
|--------|--------------|
| Welcome | App load |
| Questionnaire | Start / Continue |
| Report | Assessment complete / restore |
| مرکزی صفحہ | Welcome, report, timeline toolbar |
| ترقی کا وقت خط | Welcome, home, progress, report |
| سفرِ ترقی | Welcome, home, timeline, report |

## Data Boundaries

- **Assessments** affect scores and history snapshots
- **Journal** is personal notes only — never affects scores
- **Intelligence** reads history only — never recalculates reports
