| Before | After |
|--------|-------|
| `Firebase auth pending` | `Development mode` (when backend sync disabled) or `Authentication enabled` / `Authentication not configured` |

| `Configure Firebase for this project…` | `Authentication is not configured for the current backend provider.` |
| `Firebase integration is not active yet.` | Removed from settings intro |

Developer setup text on the backend-required screen retains `USE_BACKEND_SYNC` and `BACKEND_PROVIDER` (implementation flags, not user branding).

Internal API names (`renderAdminFirebaseRequired`, `initializeAdminFirebase`, sample data key `settings.firebase`) were **not** renamed.
