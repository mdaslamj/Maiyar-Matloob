## Layout Changes

1. **Compact header** — Question counter + percentage on one row; 6px progress bar below
2. **Tighter spacing** — Reduced padding/gaps across shell, card, answers
3. **Question card flex column** — Progress content + answers + nav fit one viewport
4. **Navigation inside card** — Previous / Next at bottom of question card
5. **Vertical answers retained** — No 2×2 grid; options min-height 48px
6. **Explanation** — Collapsed by default; compact toggle label
